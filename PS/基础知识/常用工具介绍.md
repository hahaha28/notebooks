# 常用工具介绍

* 放大，缩小

    ```note
    Alt + 鼠标滚轮
    ```

* 抓手

    ```note
    空格（不放)
    ```

* 画笔

    ```note
    b 		//切换到画笔
    [		//缩小画笔
    ]		//放大画笔
    ```

* 选择

    ```note
    v
    ```

* 颜色

    ```note
    x	// 切换
    D	// 复位
    ```

* 切换屏幕模式

    ```note
    F	// 切换
    ESC		//退出全屏
    ```
