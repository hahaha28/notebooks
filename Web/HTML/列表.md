# 列表



## 无序列表

列表项没有顺序级别之分

```html
<ul>
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
</ul>
```

**注意：**

* `<ul></ul>` 之间只能嵌套 `<li></li>` 。
* `<li></li>` 相当于一个容器，可以容纳所有元素。



## 有序列表

了解就行

```html
<ol>
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
</ol>
```



## 自定义列表

```html
<dl>
    <dt>名词1</dt>
    <dd>名词1解释1</dd>
    <dd>名词1解释2</dd>
    ...
    <dt>名词2</dt>
    <dd>名词2解释1</dd>
    <dd>名词2解释2</dd>
    ...
</dl>
```
